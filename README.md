# Macdonalds-Parser
Tesk Task - Junior Full Python Developer
## Installation
1. Clone repository
    ```bash 
    git clone https://github.com/andrii-zabolotin/Macdonalds-Parser.git
    ```
2. Change directory
    ```bash
    cd Macdonalds-Parser
    ```
3. Build docker images
    ```bash
    docker-compose build
    ```
## Run

Run all services
```bash
docker-compose up
```
